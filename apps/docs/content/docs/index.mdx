title: Build printable documents with React

---

Useprint is a React-first toolkit for building printable documents as actual React components.

The core flow is simple:

1. Build your document with `@useprint/components`
2. Preview it locally with `@useprint/cli`
3. Render it to HTML with `@useprint/render`
4. Turn that HTML into a PDF with `useprint-js` or your own Chromium setup

## Recommended Packages

For most projects, start with the authoring and preview combo:

```bash
npm install @useprint/components
npm install -D @useprint/cli
```

When you need lower-level control, add the dedicated packages directly.

## Why Useprint

- Author documents in React instead of string templates.
- Keep layout logic close to the component tree that produces it.
- Preview documents locally while you iterate.
- Render portable HTML that works with your own PDF backend or `useprint-js`.
