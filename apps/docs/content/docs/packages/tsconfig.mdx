The `tsconfig` package contains shared TypeScript presets used across the workspace.

## Included presets

- `base.json`
- `react-library.json`
- `nextjs.json`

## Should you use it outside the repo?

This package is mainly here to keep the monorepo consistent. It is not part of the main public Useprint runtime story.
