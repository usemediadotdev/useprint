# @useprint/shared

Shared constants and utilities used by the Useprint packages.

## Exports

- `PAGE_SIZES`
- `PPI`
- `MM_PER_INCH`
- `MM_TO_PX`

## Who is this for?

Most users will not need this package directly. It is mainly useful when you want the same page-size constants and unit conversion helpers that the rest of Useprint uses internally.
